# 项目简介

## 核心功能
- 基于YOLOv8的烟头检测系统
- 支持2200张烟头数据集（1类）
- 提供完整源码、标注数据集、训练好的模型及训练结果
- 包含项目运行教程
- 支持图片、视频及摄像头进行检测
- 支持选择模型
- 界面实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并查看所有检测结果
- 支持检测结果保存
- 支持选择单张图片或图片文件夹进行识别
- 支持选择视频文件进行识别
- 支持调用本地摄像头进行识别
- 支持自定义置信度和IOU阈值
- 支持选择显示标签和原图
- 支持选择检测模型
- 支持查看批量检测每一张检测结果

## 适用场景
- 环境监测与烟头检测
- 公共场所清洁管理
- 环保监测与数据分析
- 视觉检测系统开发与研究

## 技术特点
- 基于Pycharm+Anaconda或VSCode+Anaconda开发环境
- 使用Python 3.9、OpenCV-Python、PyQt5、Ultralytics、Torch1.9等技术栈
- 提供完整程序文件、UI界面源文件、图标、数据集图片和项目运行教程
- 配置成功环境后可直接使用
